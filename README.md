# Introduction

Elements is a C++/Python build framework. It helps to organize the software
into modules which are gathered into projects.


## Build Instruction

These are the standard CMake build instruction.

```
> mkdir build
> cd build 
> cmake -DCMAKE_INSTALL_PREFIX=/usr ..
> make 
> make install
```
